Lambda表达式

Lambda 表达式是一种简洁的函数表示方式，它可以包含表达式和语句。Lambda 表达式使用=>运算符，该运算符读作“goes to”。

基本语法#

Lambda 表达式的基本语法如下：

(parameters) => expression-or-statement-block

例如，以下是一个简单的 Lambda 表达式：

x => x * x

这个表达式表示一个接受一个整数参数并返回其平方的函数。

使用 Lambda 表达式#

你可以在代码中的许多地方使用 Lambda 表达式。例如，以下代码使用 Lambda 表达式来定义一个委托：

Func<int, int> square = x => x * x;

然后，你可以像调用普通函数一样调用这个委托：

int result = square(5); // 结果为25

主要用于以下几种情况#

  1. 委托和回调：Lambda 表达式可以用作委托和回调。
Func<int, int> square = x => x * x;

然后，你可以像调用普通函数一样调用这个委托。

  1. LINQ 查询：在 C#中，你可以在 LINQ 查询中使用 Lambda 表达式。
int[] numbers = { 2, 3, 4, 5 };
var squaredNumbers = numbers.Select(x => x * x);

  1. 事件处理器：当你需要提供事件处理程序时，Lambda 表达式可以非常有用。

  2. 异步编程：你可以在任何需要委托类型或表达式树的代码中使用 Lambda 表达式，例如作为 Task.Run(Action) 方法的参数，以传递应该在后台运行的代码。

结论#

Lambda 表达式是 C#中的一个强大工具，它可以让你以更简洁的方式编写函数和方法。
