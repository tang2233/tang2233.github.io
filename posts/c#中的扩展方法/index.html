C#中的扩展方法

什么是扩展方法？#

扩展方法是一种特殊的静态方法，但我们可以像实例方法一样调用它们。这使我们能够向现有类型添加新的方法，而无需修改、继承或重新编译原始类型。

如何创建扩展方法？#

  1. 静态类：扩展方法必须在静态类中定义；
  2. 静态方法：扩展方法必须是静态方法；
  3. 第一个参数：扩展方法必须以关键字 this 修饰的第一个参数开始，该参数定义了被扩展的类型。
public static class StringExtension
{
    public static string ToLowerFirstLetter(this string str)
    {
        if (string.IsNullOrEmpty(str) || char.IsLower(str, 0))
            return str;

        return char.ToLowerInvariant(str[0]) + str.Substring(1);
    }
}

在上述代码中，我们为 string 类型添加了一个名为 ToLowerFirstLetter 的扩展方法，该方法将字符串的首字母转换为小写。

如何使用扩展方法？#

string str = "Hello, World!";
string result = str.ToLowerFirstLetter();

在上述代码中，我们像调用实例方法一样调用了扩展方法 ToLowerFirstLetter。
